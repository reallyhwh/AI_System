## 技术要点

### Dify API 调用

```typescript
// 阻塞模式调用
const response = await fetch('/api/dify/v1/chat-messages', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    inputs: {},
    query: userInput,
    response_mode: 'blocking',  // 阻塞模式
    user: 'user-' + Date.now(),
  }),
});
```

### Vite 代理配置

```typescript
// vite.config.ts
server: {
  proxy: {
    '/api/dify': {
      target: 'https://api.dify.ai',
      changeOrigin: true,
      rewrite: (path) => path.replace(/^\/api\/dify/, ''),
    },
  },
}
```
