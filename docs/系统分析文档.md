## дёҖгҖҒзі»з»ҹеҲҶжһҗж–№жі•и®ә

### 1.1 еҲҶжһҗж–№жі•

#### 1.1.1 иҮӘйЎ¶еҗ‘дёӢеҲҶжһҗ

```
дёҡеҠЎзӣ®ж Ү вҶ’ дёҡеҠЎжөҒзЁӢ вҶ’ еҠҹиғҪжЁЎеқ— вҶ’ ж•°жҚ®з»“жһ„ вҶ’ д»Јз Ғе®һзҺ°
```

**жӯҘйӘӨиҜҙжҳҺпјҡ**
1. **дёҡеҠЎзӣ®ж Ү**пјҡзҗҶи§Јзі»з»ҹиҰҒи§ЈеҶід»Җд№Ҳй—®йўҳ
2. **дёҡеҠЎжөҒзЁӢ**пјҡеҲҶжһҗзі»з»ҹж”ҜжҢҒе“ӘдәӣдёҡеҠЎеңәжҷҜ
3. **еҠҹиғҪжЁЎеқ—**пјҡжӢҶи§Јзі»з»ҹзҡ„еҠҹиғҪиҫ№з•Ң
4. **ж•°жҚ®з»“жһ„**пјҡзҗҶи§Јж ёеҝғж•°жҚ®еҸҠе…¶е…ізі»
5. **д»Јз Ғе®һзҺ°**пјҡж·ұе…ҘжҠҖжңҜз»ҶиҠӮ

#### 1.1.2 4+1 и§ҶеӣҫжЁЎеһӢпјҲIEEE ж ҮеҮҶпјү

| и§Ҷеӣҫ | е…іжіЁзӮ№ | еҜ№еә”и§’иүІ | жң¬йЎ№зӣ®еә”з”Ё |
|------|--------|----------|------------|

| ејҖеҸ‘и§Ҷеӣҫ | жЁЎеқ—еҲ’еҲҶгҖҒдҫқиө–е…ізі» | зЁӢеәҸе‘ҳ | зӣ®еҪ•з»“жһ„гҖҒеҢ…дҫқиө– |
| иҝӣзЁӢи§Ҷеӣҫ | иҝҗиЎҢж—¶иЎҢдёәгҖҒе№¶еҸ‘ | зі»з»ҹйӣҶжҲҗиҖ… | Dify API и°ғз”ЁжөҒзЁӢ |

| еңәжҷҜи§Ҷеӣҫ | ж ёеҝғз”ЁдҫӢгҖҒдёҡеҠЎжөҒзЁӢ | жүҖжңүи§’иүІ | еҮәе·®з”іиҜ·гҖҒе·®ж—…жҠҘй”ҖжөҒзЁӢ |
